# Ревью #1 — Codex T1-T8 (Fable), коммит 7a58e9c

**Вердикт: GO с правками.** Каркас цел, антивраньё реализовано по дизайну буквально, 134 теста честно зелёные. Найдены 2 дыры в ядре антивранья + границы эскалации + хвосты frunze. Правки уходят Codex (см. коммит с фиксами).

## MAJOR
- **M1** `validator.py` — детекторы балла/скидки слепнут, когда LLM пишет через « — » (нормализация тире → «. » рвёт контекстное окно детектора). «Проходной балл — 90» не ловится. → гонять факт-детекторы до замены тире + регресс-тесты.
- **M2** `faq.py` seed — кыргызские паттерны отвечают по-русски (нарушение правила языка). → убрать ky-паттерны, ky-вопросы отдать LLM.
- **M3** `faq.py` — паттерн «оплата» в правиле цены перехватывает вопрос, который обязан идти менеджеру. → удалить паттерн.
- **M4** `faq.py` — «после 11» перехватывает спорный вопрос о сроке без эскалации. → negative_terms на длительность.
- **M5** `docker-compose*.yml` — протёкшие profile_id frunze + BOTS scenario=tours/visa (упадёт на Literal["admission"]) + tourvisor/frunze-БД. → плейсхолдеры колледжа.
- **M6** `scripts/run_scenarios.py`, `scenarios/frunze_v1.json`, `smoke_test.py`, `seed_demo.py` — мёртвый QA-контур (tourvisor_*, /admin/board/visa). → под admission или заглушка с TODO.
